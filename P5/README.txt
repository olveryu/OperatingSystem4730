##Instructions

To clean my code(i.e remove any compiled files), type:

$ make clean

To compile my code, type:

& make

To run my code(after compiling it), type:

usage:
	./vm [options] < input_file

Options:
        -R page_replacement_policy [0 - RANDOM, 1 - FIFO, 2 - LRU, 3 - CLOCK]
        -P NUM_PROC  (default 2)
        -t TLB_ENTRY (default 4)
        -f NUM_FRAME (default 8)
        -p NUM_PAGE  (default 8)
        -g DEBUG     (default false)

-----------
        -X  (default false )
